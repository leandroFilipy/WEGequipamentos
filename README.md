## 🛠️ Sobre o Projeto

O **WEGequipamentos** foi criado para automatizar o processo de controle de equipamentos, garantindo agilidade, segurança e transparência. Com ele, você pode cadastrar novos itens, editar informações, buscar dados rapidamente e gerar relatórios sob medida. 

## ✨ Funcionalidades

- Cadastro de equipamentos 📝
- Edição e exclusão de registros 🗑️
- Busca avançada por filtros 🔍
- Visualização detalhada das informações 👁️
- Relatórios e exportação de dados 📊

## ⚙️ Instalação

1. Clone este repositório:
   ```bash
   git clone https://github.com/leandroFilipy/WEGequipamentos.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Execute o projeto:
   ```bash
   npm start
   ```

## 🚦 Como Usar

Após a instalação, acesse `http://localhost:3000` no navegador. Utilize o menu para navegar entre as funcionalidades do sistema e aproveite todos os recursos! 😃
